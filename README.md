# LibraryHub — Library Management Software

A web-based Library Management System built with HTML and CSS. It allows librarians to manage books, members, and issue/return operations through a clean, responsive dashboard interface.

## Features

- **Login Page** – Secure login screen to access the management dashboard.
- **Dashboard** – Overview of key library statistics (total books, issued books, overdue books, active members), monthly issue trends, genre breakdown, and recent activity feed.
- **Books** – Browse and manage the library's book catalogue.
- **Members** – View and manage registered library members.
- **Issue / Return** – Track book issues and returns.

## Project Structure

```
├── index.html        # Login page
├── dashboard.html    # Main dashboard
├── books.html        # Books catalogue page
├── members.html      # Members management page
├── issues.html       # Issue / Return page
└── css/              # Stylesheets (see css/README.md)
```
